# git-practice

A practice repository for learning Git and GitHub.

## What I learned
- How Git tracks changes using commits
- How branches work as pointers
- How to push a local repository to GitHub

## Commands reference
- `git init` — create a repository
- `git status` — see what's changed
- `git add` — stage files
- `git commit -m "message"` — create a checkpoint
- `git push` — send commits to GitHub
